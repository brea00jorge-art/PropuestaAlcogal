# 🏭 Alcogal Soluciones de Limpieza y Protección Industrial B2B

Bienvenido al repositorio de nuestra plataforma web renovada. Somos especialistas con más de **30 años de experiencia** en la elaboración, comercialización y distribución de trapos industriales, productos de higiene y seguridad laboral de alto rendimiento.

## 🚀 Sobre el Proyecto
Esta web ha sido rediseñada para ofrecer una experiencia de usuario (UX) moderna y funcional, alineada con las necesidades de sectores críticos como el **eólico, naval, metalúrgico, pesquero y automoción**.

### ✨ Características Principales
- **Interfaz Moderna:** Nueva paleta de colores vanguardista y tipografía optimizada para lectura técnica.
- **Catálogo Especializado:** Clasificación detallada de productos según necesidades técnicas.
- **Enfoque B2B:** Estructura orientada a la eficiencia y el servicio profesional.

---

## 🛠️ Líneas de Negocio
Nuestra oferta se diferencia en cinco pilares fundamentales:

1.  **Protección y Seguridad Laboral:** Equipamiento de marcas líderes con la mejor relación calidad/precio.
2.  **Higiene y Mantenimiento Industrial:** Trapos de limpieza técnica, absorbentes y celulosas fabricados con tecnología innovadora.
3.  **Manutención y Almacenaje:** Soluciones logísticas para la industria.
4.  **Líneas de Vida y Sistemas Anticaídas:** Seguridad en altura certificada.
5.  **Servicios Especializados:** Cursos de formación y revisión de equipos de elevación.

---

## 🎨 Diseño y Estilo
El proyecto utiliza una identidad visual renovada:
